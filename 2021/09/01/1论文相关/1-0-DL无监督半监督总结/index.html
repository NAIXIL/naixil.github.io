综述

  • 分类：
    • 半监督分类
      • 数据由有标记的实例和无标记的实例的训练数据集构成，其目标是同时从有标记的和无标记的数据训练分类器
    • 半监督聚类
      • 训练数据集由未标记的实例和关于聚类的监督信息构成，其目标是获得比单独从无标记数据聚类更好的聚类
    • 半监督回归
      • 给定一个包含有标记的实例和无标记实例的训练数据集，其目标是从一个单独的带有标记数据的回归算法改进回归算法的性能，该回归算法预测实值输出而不是类

CResST-CVPR-2021

  1. 常见长尾分布的解决
    1. 方法
      1. 重采样（Re-Sampling）
        1. 方法
          1. 对头部类别（数量多的样本）欠采样
          2. 对尾部类别（数量少的样本）过采样/重复采样，增加其样本数
        2. 缺点：模型对头部数据欠拟合，对尾部数据过拟合
      2. 重加权（Re-Weighting）
        1. 核心思想：将类别少的样本赋予更大的权重，类别多的样本赋予更少的权重
  2. 文章解决的问题：半监督场景下的长尾分布问题。没有足够标记的样本，且有标记样本的分布还是长尾分布的（类别不平衡的）。
  3. 用FixMatch做了实验，表明该模型对于头部样本的recall很高，对尾部样本的Recall很低；对尾部样本的Precision很低，对尾部样本的Precision很高。表明：对头部样本过拟合，对尾部样本欠拟合
  4. 本文方法：
    1. 使用标准的SSL算法利用（已标记数据集和未标记数据集的信息）训练一个有效的模型；
    2. 用训练出来的模型给（未标记数据集$\mu$）中的每个样本打上伪标记得到（新的数据集$\hat\mu$）
    3. 挑选出模型的预测类别属于尾部类别的样本作为候选集$\hat s$加入到已标记集合中
      • 挑选的方式：类别越少，入选概率越大。
  5. 评价：
    • 第三步里面，模型预测的类别属于尾部，意味着样本的伪标记具有很高的置信度（$High\ precision$），因为此时模型是对头部类别过拟合，此时模型还将样本预测成尾部说明该伪标记真的是真实情况。
    • ==与active learning + self-paced learning思路类似，可以参考一下==

伪标签技术（$pseudo-labled\ data$）

  • 论文：Pseudo-Label : The Simple and Efficient Semi-Supervised Learning Method for Deep Neural Networks
  • 原理：利用在已标注数据所训练的模型在未标注数据上进行预测，根据预测结果对样本进行筛选，再次输入模型中进行训练的过程。
  • 应用条件：
    • 满足聚类假设：相同类的示例有较大可能拥有相同的标记
    • 熵正则化
  • 方法1
    1. 使用标记数据训练有监督模型M
    2. 使用M对无标签数据进行预测，得出概率P
    3. 通过P筛选高置信度样本
    4. 使用有标记数据以及伪标签数据训练新模型M’
  • 方法2
    1. 使用标记数据训练有监督模型M
    2. 使用M对无标签数据进行预测，得出概率P
    3. 通过P筛选高置信度样本
    4. 使用有标记数据以及伪标签数据训练新模型M’
    5. 将M替换成M‘，重复2~4
  • 方法3：
    1. 使用标记数据训练有监督模型M
    2. 使用M对无标签数据进行预测，得出概率P
    3. 将模型的损失函数改成$Loss=loss(labeled_data)+alpha*loss(unlabled_data)$
    4. 使用有标记数据以及伪标签数据训练新模型M’

什么是正则化（Regularization）

  1. 定义：因为神经网络拟合能力比较强，容易造成过拟合，降低了泛化能力。正则化是一类通过限制模型复杂度，从而避免过拟合，提高泛化能力的方法。
  2. 方法：
    1. 引入一些约束规则
    2. 增加先验
    3. 提前停止
    4. 丢弃法（Dropout）
      1. dropout是给全连接层用的
      2. BatchNormalization是给卷积用的
