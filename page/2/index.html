基础概念

分位数（Quantile），亦称分位点，是指将一个随机变量的概率分布范围分为几个等份的数值点，常用的有中位数（即二分位数）、四分位数、百分位数等。

四分位数（Quartile）是统计学中分位数的一种，即把所有数值由小到大排列并分成四等份，处于三个分割点位置的数值就是四分位数。

1）第一四分位数(Q1)，又称“较小四分位数”，等于该样本中所有数值由小到大排列后第25%的数字；

2）第二四分位数(Q2)，又称“中位数”，等于该样本中所有数值由小到大排列后第50%的数字；

3）第三四分位数(Q3)，又称“较大四分位数”，等于该样本中所有数值由小到大排列后第75%的数字。

第三四分位数与第一四分位数的差距又称四分位距。

  • 校园网概况
    • 校园网是一个大局域网，理论上连上校园网的所有主机都能够相互ping通（在二者都具有101.7.xxx.xxx网段的IP的情况下）。
    • 平时连接校园网的两台主机无法建立TCP远程连接（也就是没法直接远程桌面），原因有2：
      • 两主机跨局域网，且需要被远程的主机IP为私有IP（即10.0.0.0-10.255.255.255 、172.16.0.0—172.31.255.255 、192.168.0.0-192.168.255.255 三个网段）
      • 校园网屏蔽了3389端口
    • 信息与计算机学院网段分布：101.7.186.xxx为实验楼的网段；101.7.187.xxx为北楼的网段（例如607打印机的IP就是101.7.187.235）
  • 本文适用
    • 只要使用人能连上校园网（在家或宿舍可以用motion pro通过VPN连接校园网http://iadmin.tyut.edu.cn/info/1155/2280.htm）就能连接上学校的任意已联网服务器，且连接质量远好于向日葵或FRP。
  • 远程连接的方法概览（都是在需要被远程连接的主机上操作）

    • 首先要保证服务器主机有一个101.7.xxx.xxx网段的IP，上网方式不要设置成DHCP，设置成静态IP。具体到407，就是101.7.186.xxx网段的IP。

      • 设置静态ip相关信息：

        • IP地址：101.7.186.XXX # 除了128、126、125、1、200都可以试试
        • 子网掩码：255.255.255.0
        • 默认网关：101.7.186.1

      • DNS为202.207.247.227、202.207.240.225

      • 20210728新增注意：服务器需要登录上自己的ipv6账号，如果插上网线就能上网不需要经过登录的话一般是连接在同一交换机的其他服务器有人登录了，此时自己的服务器仍然是处在小局域网而非校园网这个大局域网下。如ip为101.7.187.123，不经过ipv6登录的情况下，只有当自己电脑的ip是101.7.187.xxx的时候才能连上，而非初衷101.7.xxx.xxx就能连接上。

    • 修改服务器主机的远程端口为2000 ~ 3388、3390 ~ 65535之间的任意值，教程见下文。

    • PS：推荐能连接校园网的时候用此方法，连不上校园网的时候自己搭frp或向日葵等。

      • 对比：
        • 连接质量：修改端口>自建frp>第三方软件
        • 稳定性：修改端口=第三方>自建frp

逻辑判断（判断5、6、7、8，9h）

翻译推理

  • 要素

    • 前推后
      • 若……则……；只要……就……；所有……都……；为了……一定（必须）……；……是……的充分条件
        • 如果想要取得好成绩，就必须减少玩耍时间 好成绩→少玩耍
          • 取得好成绩 肯前
          • 没取得好成绩 否前
          • 减少玩耍时间 肯后
          • 增加玩耍时间 否后
    • 后推前
      • 只有……，才……；不……，不……
        • 人无精神则不立 立$\rightarrow$精神
      • 除非 A 否则不 B：B→A；除非 A 否则 B：-B→A
        • 除非天地合，否则我不会与君绝
        • 除非交房租，否则搬出去。不搬出去$\rightarrow$交房租
      • 谁必不可少，谁在箭头后
        • …是…的基础/假设/前提/关键/必要/必不可少条件
      • A的充分条件是B。B$\rightarrow$A
      • A的必要条件是B。A$\rightarrow$B

  • 规则：“逆否等价”

    • $A\rightarrow B=-B\rightarrow -A$
    • 肯前必肯后，否后必否前；否前肯后不知道
    • 注意：
      1. 选项与题干无关，就不选，如“坚持锻炼→风险降低”，选项“是否坚持锻炼，直接决定了阿尔茨海默症等神经退行性疾病的发病风险能降低”就不选
      2. 递推关系可传递
        • 1.“A→B”和“B→C”，可以写为“A→B→C”。
          2.“A→B”和“-A→C”，后者逆否为“-C→A”，与前者串联为“-C→A→B”。
        • 3.“A→B”和“C→-B”，后者逆否为“B→-C”，与前者串联为“A→B→-C”。
        • 4.“A→B”和“A→C”不能串联。
        • 5.“A→B→C”可以写为“A→C”，也可以写为“-C→-B→-A”；“A→B→-C”
          可以写为“C→-B→-A”，均可以随意变换。

  • 要素

    • 且
      • 和、既……又……、不仅……而且……、……但是……
    • 或
      • 或者、或者…或者…、至少一个（两个也行）
        • 否1$\rightarrow$1
          • 我不看篮球或不看足球。看篮球$\rightarrow$不看足球
      • AB至多一个。-A或-B
    • 德摩根定律
      • -（A 且 B）=-A 或-B
      • -（A 或 B）=-A 且-B
      • “-”进去，“且”“或”互变
        • -（方便面且大盘鸡）=-方便面或-大盘鸡
        • -（咖啡因或泡枸杞）=-咖啡因且-泡枸杞
  • 要素
    • 提问：以下哪项与题干中推理形式/结构相同？
    • 把名词换成字母

组合排列

  • 题型特征：题干中给出一组对象和相关信息，要求把对象和信息进行匹配
  • 方法
    • 排除法
      • 何时用：题干信息确定
      • **首要确定谁不是谁 **
        • 比大小的题目，极值很重要。（谁最大，谁最小）
    • 代入法
      • 假设选项正确，代入题干
      • 题干有“补充/添加以下哪项可以退出”，尝试代入
      • 设问中有可能、不可能，考虑代入
        • 隧道排序问题
  • 推理技巧
  1. 优先关注出现次数最多信息
    • -
    • - 根据题干，只有一个人考上了研究生，只有一个人说的正确且说的正确的人考上了研究生，可以知道，研究生只有一个且说的话正确；从“研究生”出现的地方入手，G说的话，必是假，因为如果G对，那么G是研究生，又因为研究生只有一个，所以G假话；可以推知H肯定没考上研究生，也就是H说了假话，也就是J是公务员，所以选D。
  2. 符号“>”、“<”。往往涉及年龄、成绩、身高、收入等大小比较
  3. 对象和信息比较多的时候，列表格；排序题考虑间隔

逻辑论证

  1. 论点提示词：所以，结论是，这表明/说明/意味着，由此推出/可知，据此
    认为
  2. 论据提示词：由于、因为、鉴于、根据
  3. 论据常见形式：原因、数据、事例、实验或调查内容等
  • 削弱题型：反对
    • 题型特征：提问“削弱”、“质疑”、“反驳”、“否定”
    • 削弱方式：论点与论据一致，优先考虑否定论点；论点与论据不一致、没有否定论点的选项、提问方式为削弱论证的时候，优先考虑拆桥。
      • 否定论点
        • 选项特征：与论点表述的意思相反
        • 解题步骤
          1. 找出论点
          2. 想出与论点意思相反的表述
          3. 寻找对应选项
        • 例子
          • - 但是之前的不看，只有论点，否定论点：养老机构的噪音不影响居民生活。对比选项，A中心在位置，B中心在不了解，D中心在办养老机构之前，都无关
        • 常见的论点特殊形式
          • 论点为“不是 A，是 C 导致 B”：甲：A 导致 B;乙：不是 A，是 C 导致 B
            • 选项就找常考削弱乙——A 导致 C
      • 拆桥
        • 题型特征说论点与论据之间没有必然的联系。
          • 特殊考法：论点为某物质比较好，论据为物质中的某些元素好
            • 解题方法：元素含量少，没啥作用
      • 否定论据
      • 因果类
        • 什么是因果论证？
          • 1.论点包含因果关系
            • ……是……的原因、
            • ……导致/使得/有助于……
            • ……增加/降低/加强/减轻……
              • 例：研究人员认为，“缺觉”是导致第二天没有精神的原因
              • 例 ：有研究小组调查显示，在封闭环境中入睡的人更容易频繁醒来，而且自我感觉睡眠质量不佳。因此，有人认为，就寝时关闭门窗将会降低睡眠质量
          • 因果倒置削弱：原因与结果说反
          • 他因削弱：承认结果，同一个主体存在两种及以上可能的原因；如果论点是其中一种原因，则可以用另一种原因进行削弱。
            • 他因的特点：与论点是同一个主体、同一时间、不同原因
  • 加强提醒：赞同
